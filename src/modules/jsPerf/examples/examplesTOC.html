jsPerf is a simple-to-use JavaScript module that measures memory and speed. It works both in node and on the web and comes with several examples. jsPerf also includes a real-time memory widget so you can observe overall memory behavior.

memory examples

For memory measurements, be sure that you are either using 
webExamples/runWebExamples.{bat|sh}
or are running a local web server on port 8000 and chrome with the correct memory flags. For example: 

chrome --js-flags="--expose-gc" --enable-precise-memory-info        \
  "https://localhost:8000/src/modules/jsPerf/examples/webExamples/  \
    webContinuousMemoryExample/webContinuousMemoryExample.html"

jsPerf widget example

The jsPerf widget is js/css that you can easily include in your app. It will create a small container that shows performance metrics. You will still need to run with the chrome flags, but it makes it easy to monitor and measure application performance. 

node ../../../../tools/server/simpleServer.js --webroot "../../"
chrome --js-flags="--expose-gc" --enable-precise-memory-info \
  "https://localhost:8000/src/modules/jsPerfWidget/examples/jsPerfWidgetExample.html"
