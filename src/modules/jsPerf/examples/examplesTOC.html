jsPerf is a simple-to-use JavaScript module that measures memory and code execution speed.   JsPerf is open source on github

For memory measurements, be sure that you are either using 
webExamples/runWebExamples.{bat|sh}
or are running a local web server and chrome with the correct memory flags. For example:

chrome --js-flags="--expose-gc" --enable-precise-memory-info "https://localhost:8000/... .../webSimpleTimerExample.html"
